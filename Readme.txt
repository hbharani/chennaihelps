Install Python
install virtualenv
virtualenv disasterhelp 
pip install flask
pip install flask-pymongo
pip install flask-wtf (not sure if i did this or it comes with flask)


copy all the files inside disasterhelp.zip into disasterhelp folder

change local ip address to your machine address in app.run("192.168.0.3",debug=True)

python run.py (inside disasterhelp folder)


this should run the app in your local machine for you to test.
